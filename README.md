# Bright-Coffee-Shop-Case-Study
Data analysis on sales performance for the last 6 months with insights for CEO

## Problem Statement
The shop is can no longer carry its costs,transactions are happening daily ,with big DATA available on site there is a need for quantitative analysis.
I am tusk to do tracking for sales daily and monthly basis adn find the bottleneck on the system ie time taken to close a transaction.

## Aim
The aim of the project is to analyze the coffee shop sales decline with evidence base information on excel for deep diving reporting 
and on dashboard or PowerBI for continues reporting that wiil assist for future predictions. 

## Objectives
To be able to Analyze transaction trends with quantitative data and visualize daily,weekly and monthly transactions.Insights on peaks hours and offpeak hours for resource ie people on shift,product on demand and time taken to close a 1 transaction.Lates trends if there is a need
to adjust the menu for a particular season.Implement a strategy ie Low cost strate bieng economy of scale,differentiation or value creation 
strategy.
